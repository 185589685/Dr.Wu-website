本教程基于 ROS2 Foxy，Ubuntu20.04。

## 环境安装

参考文章 [**ROS2 入门教程 ——2. Ubuntu20.04 安装 ROS2 Foxy**](https://www.guyuehome.com/10226)。

卸载：

```
sudo apt remove ros-foxy-* && sudo apt autoremove

```

检查～/.bashrc 　以及／opt / 目录是否有 ros 文件夹存在。
